DOM Elements

React 为了性能和跨浏览器兼容实现了一个独立的DOM 系统。我们有机会去除一些粗糙的浏览器实现。

在React 中，所有的DOM 属性（properties）和特性（attributes）（包括事件句柄）都应该是camelCased。例如，HTML 特性tabindex 对于React 特性tabIndex。特例是aria-* 和data-* 特性应该是lowercased。
