过渡

当元素从一个状态更改为另一个状态时，将在浏览器中进行过渡。浏览器自动在每个状态之间绘制帧以创建移动。

过渡transition 是CSS 中的属性。就像你给一个元素一个高度，宽度或边框一样，我们也给出了元素过渡。

注意过渡属性为CSS 状态中的一个按钮引用。这告诉浏览器将过渡应用于任何状态更改，例如在悬停时以及从悬停状态更改时。

如果我们仅将过渡属性应用于悬停状态，它只会转换到悬停状态但不会返回。
