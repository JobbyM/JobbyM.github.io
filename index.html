响应式设计

早年设计 Web 时，页面是以适配特定的屏幕大小为考量创建的。如果用户正在使用比设计者考虑到的更小或者更大的屏幕，会导致有多余的滚动条、过长的行以及没有被合理利用的空间。由于屏幕尺寸的种类越来越多，出现了响应式网页设计的概念（responsive web desgin, RWD），RWD 指的是允许 Web 页面适应不同屏幕宽度，进行布局和外观的调整的一系列实践。
