Getting Start

webpack is a tool to build JavaScript modules in your application

Creating a bundle

创建一个demo 目录去尝试webpack。安装webpack

现在创建一个index.js 文件
app/index.js

需要下面的html 文件来运行上面这段代码
index.html

在这个例子中，在script 标签中包含了依赖。
index.js 依赖 lodash 运行。
为了在index.js 文件中使用lodash，我们只需要引入lodash.
app/index.js

同时我们需要修改index.html

现在index.js 明确的需要 lodash，绑定_（没有全局污染）。
现在项目文件夹运行webpack，通过入口文件index.js 生成’bundle.js’ 文件。

Using webpack with a config

对于更复杂的配置，我们可以使用配置文件
webpack.config.js

通过webpack 来运行

如果webpack.config.js 文件存在，webpack 命令默认使用此配置文件

Using webpack with npm

通过调整package.json 文件来简化使用webpack 命令

同时使用npm run build 命令实现webpack 功能
