添加显示隐藏Form 表单的功能

功能描述：

  1. 点击“添加问题”显示表单
  2. 点击表单上的“取消”按钮隐藏表单

功能实现：

  1. 利用this.state 进行实现，添加一个事件onToggleForm
  2. QuestionApp.js 文件中进行初始化state， 同时提供onToggleForm 方法 在ShowAddButton 和QuestionForm 组件上添加组件属性
  3. ShowAddButton.js 也需要添加相应的属性
  4. QuestionForm.js 上也要添加属性 同时需要增加一个style

提供默认的列表数据进行展示

  1. QuestionApp.js 中提供数据this.state.questions，将数据传入QuestionList 组件
  2. QuestionList.js 中接受this.props.questions 进行数据的处理
  3. QuestionItem.js 中将每一个数据展示在对应的位置上

实现通过form 表单进行数据的添加

  1. QuestionApp.js 中提供onNewQuestion, 将其传入QuestionForm 组件
  2. QuestionForm.js 中处理onSumit 事件，在此事件方法中调用onNewQuestion 方法

实现对点赞的事件处理

  1. QuestionApp.js 中提供onVote 同时将此方法传入QuestionList 组件
  2. 在QuestionList 组件中，将其传入QuestionItem 组件
  3. 在QuestionItem 组件中，在voteUp 方法中进行调用 在button 按钮上添加voteUp 事件
