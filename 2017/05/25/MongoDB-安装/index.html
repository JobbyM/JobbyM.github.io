什么是MongoDB

MongodB is a document database with the scalability and flexibility that you want with the querying and indexing that you need

MongoDBwww.mongodb.com/what-is-mongodb

安装

  1. 首先到官网 进行下载，在百度云盘提供了一版
  2. 将下载的.msi 进行安装
  3. 进入安装目录
  4. 启动服务，进入bin目录，运行mongod 提示如图 根据install-mongodb-enterprise-on-windows需要新建一个data\db 目录 应该是MongoDB 命令中 不能有带空格的目录名，重新建立目录D:\data\db 这次成功了！
