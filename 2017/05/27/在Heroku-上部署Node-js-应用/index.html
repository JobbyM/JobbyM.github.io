前言

工作之余，按照nswbmw 的一起学Node.js教程，一步一步实现了一个Express + MongoDB 的多人博客，并部署到了Heroku 上。

本篇教程就是在4.15 部署的基础上，按照自己的实际操作记录的部署过程。

部署到Heorku

  2. 本地进入D:\nodejs\my-blog 添加远端仓库
  3. 提交代码
  4. 新建分支heroku，master 分支为一起学Node.js 教程的4.15 部署 之前的内容
